# FoodBankBrowser
(Free Use) A browser for food banks in the United States of America
To run on a local server or to publish to a website.

simply run index.html (on a server like xampp) or publish on a website to use.

Layout:

Path to Files on Site
├── index.html

└── Datasets 
    ├── ak_food_pantries.json
    └── al_food_pantries.json
    └── ... (other state files)
